# Bandit Lab — OverTheWire

## Descripción
Repositorio que documenta la resolución de los niveles 0–10 del wargame Bandit de OverTheWire, usando la CLI de Linux y Git para versionar paso a paso.

## Contenido
- `bandit-report.md`: Documentación por nivel con comandos y explicaciones.
- `README.md`: Introducción, herramientas, estructura y guía de uso.

## Herramientas
- SSH para conexión remota
- bash y utilidades Unix (`ls`, `cat`, `find`, `grep`, `sort`, `uniq`, `strings`, `base64`)
- Git (commits atómicos, ramas, push)

## Instrucciones para reproducir
1. Clonar el repositorio.
2. Leer `bandit-report.md`.
3. Usar los comandos indicados para replicar localmente o entender cada paso.
